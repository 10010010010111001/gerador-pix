FAQ

O site armazena meus dados bancários?

Não! Não armazenamos em nenhum momento! Não armazenamos os dados informados como CPF, Telefone. Não armazenamos nenhum dos dados gerados! Os dados são processados pelo servidor e retornados sem armazenar qualquer dado sensível

Esse site utiliza cookies?

Privacidade e o cookie _cfduid

O cookie _cfduid coleta e torna anônimos os endereços IP do usuário final usando um hash unilateral de determinados valores para que não possam ser identificados pessoalmente. O cookie é um cookie de sessão que expira após 30 dias. O cookie _cfduid não:
-> permitir o rastreamento entre sites.
-> siga os usuários de site para site mesclando vários identificadores _cfduid em um perfil ou corresponda a qualquer ID de usuário em um aplicativo da web do Cliente.

Isenção de responsabilidade

Os QR codes não foram testadas em todos os bancos, antes de transferir verifique se as informações estão corretas. O site se isenta de qualquer responsabilidade pela exatidão e integridade das informações divulgada.

Mantenha sempre seu computador atualizado e verifique o antivirus, se notar que alguma imagem foi alterada verifique seus programas instalados.

Tem API para gerar o QRCode no meu sistema?

Oficialmente o site não fornece mas caso queria utilizar por sua conta e risco, segue a chamada:


curl 'https://www.gerarpix.com.br/emvqr-static' \
                    -H 'Content-Type: application/json;charset=UTF-8' \
                    --data-binary '{"key_type":"Telefone","key":"9117bedf-de7d-4a4d-b55b-fc9ff06a8067","name":"FELIPE BARRETO BERGAMO","city":"São Paulo","amount":"R$ 5,00","reference":"CAFE Felipe"}'
